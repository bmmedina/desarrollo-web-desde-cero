Para construir una tabla utilizamos la etiqueta table como contenedor global, con el modificador border, damos el marco que lo rodea. Para establecer los encabezados usamos la etiqueta tr + la etiqueta th. El encabezado general para todos los campos se construye introdu ciendo el modificador coolspan +comillas y un valor, siendo este valor el numero de columnas que compartiran encabezado. despues del modificador cerramos abertura de th y a continuacion ponemos la etiqueta de cierre. Para los encabezados individuales de cada campo, usamos de nuevo la etiqueta tr + th por cada encabezado que queramos poner . Finalmente para cada fila (conjunto de campos) el proceso es el mismo, crear un bloque tr+th por cada fila, siendo el numero de trs el numero de filas y el numero de th el numero de campos de cada fila.
